Site Plan: FinScope Tools

Site Name

FinScope Tools was selected as the name because it reflects the purpose of the website: providing clear tools and scope for making better financial decisions. The name combines "Finance" and "Scope", indicating a broad and insightful view of personal investment opportunities.

Optional domain availability: finscope-tools.com

Site Purpose

The site provides users with personal finance tools and information including current investment rates, inflation data, and stock market performance such as Nasdaq and Dow Jones. It offers an interactive investment calculator and comparison tools to evaluate different investment strategies. Live financial data is fetched from free APIs to ensure accuracy and up-to-date insights.

Scenarios

  • How can I compare historical stock market returns with inflation?
  • What’s the best investment calculator for comparing multiple options over time?

Color Schema

  • Primary Color: #005f73 – Used for headings and navigation highlights.
  • Accent Color: #0a9396 – Used for links, buttons, and emphasis text.
  • Background Color: #f5f7fa – Light and clean background to improve readability.
  • Typeface Color: #2e2e2e – Dark gray for paragraph text to avoid harsh contrast.

Typography

  • Segoe UI – Used for all body text for readability and modern appearance.
  • Tahoma – Backup font for body and UI elements if Segoe UI is not available.
  • Verdana – Applied to special captions or disclaimers for legibility at smaller sizes.
